1.Python 개념

파이썬이란?

네덜란드 출신 귀도 반로섬(Guido van Rossum)이 1989년에 개발한 프로그래밍 언어

파이썬의 특징

1) 파이썬은 인터프린터 언어이다

컴파일러를 사용하는 C, Java와 다르게 파이썬은 interpreter에 의해 실행되는 스크립트

컴파일러 언어에 비해 실행속도는 느리지만 코드가 완전히 작성되지 않아도 작성된 부분까지 테스트도 가능하고 대화형 기능을 통해 곧바로 수정이 가능하다

2) 문법이 간결하고 쉽다

C, Java에서는 block을 중괄호로 구별하지만 파이썬은 들여쓰기로 구분한다.

3) 생산성이 높고 빠른 개발속도를 가지고 있다.

다른 사람이 제작한 다양한 라이브러리를 사용하여 빠르게 프로그래밍 가능

3.변수

특정한 값을 저장하는 공간

ex) myStr = “Cheer for your life”

변수 이름 규칙

  1. 첫글자는 소문자 또는 언더바(_)
  2. 영문자와 숫자, 언더바(_)로만 작성할 수 있다.
  3. 변수명은 대소문자를 구별한다.
  4. 미리정의된 예약어는 사용할 수 없다.
Python reserved word (파이썬 예약어)

변수의 데이터 타입

c언어나 java와 달리 변수 타입을 명시하지 않음.

파이썬에서는 변수를 선언할 때 별도로 타입을 명시하지 않아도 변수에 대입되는 데이터 타입에 따라 변수 타입을 지정해준다.

출력
<class ‘int’>
<class ‘Str’>

type()를 실행하면 해당 변수 타입의 class가 나오는데 이는 파이썬에서 각각의 데이터 타입이 하나의 class로 작성되어 제공하기 때문

숫자형 타입(numeric types) - int, float

정수, 실수를 가지는 데이터 타입

불리언 타입(boolean types) - bool

논리적인 값 True와 False 중 하나를 저장하는 데이터 타입

python에서는 True와 False가 예약어이므로 첫문자는 대문자로 지켜 써야한다.

C언어 에서는 1이면 참, 0이면 거짓이었지만

파이썬에서는 다음과 같이 참과 거짓을 판단한다

문자형타입(text sequence types) - str

문자열(String)은 문자로 이루어진 데이터 집합.

주로 ‘ ‘ 작은 따옴표 경우는 문자 1개 즉 char타입을 나타낼때 사용하고 “ “큰따옴표는 문자열을 나타낼때 사용하지만

파이썬에서는 char 타입이 없고 문자와 문자열을 구분하지 않으므로 어떻게 쓰던 상관이 없다.

단, 다른 언어에서 코딩이나 좋은 습관을 위해서는 구분해주는 것이 좋다

파이썬에서는 다음과 같은 방법으로 문자열을 사용 할 수 있다.
