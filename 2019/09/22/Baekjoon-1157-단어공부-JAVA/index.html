Author: P_Jinsu Date: September 22, 2019  1:43:34 Category: Baekjoon알고리즘

Baekjoon-1157

문제

알파벳 대소문자로 된 단어가 주어지면, 이 단어에서 가장 많이 사용된 알파벳이 무엇인지 알아내는 프로그램을 작성하시오. 단, 대문자와 소문자를 구분하지 않는다.

입력

첫째 줄에 알파벳 대소문자로 이루어진 단어가 주어진다. 주어지는 단어의 길이는 1,000,000을 넘지 않는다.

출력

첫째 줄에 이 단어에서 가장 많이 사용된 알파벳을 대문자로 출력한다. 단, 가장 많이 사용된 알파벳이 여러 개 존재하는 경우에는 ?를 출력한다.

예제 출력 1

예제 입력 2

예제 출력2

풀이

처음에는 char형 배열을 만들어 각 배열에 A-Z를 넣으려 하였으나 그럴 경우 해당 단어가 몇번 사용 되었는지 셀 수 없어 숫자가 증가하려면 어떻게 해야할까 고민하다 int형 배열로 A-Z개수 만큼 배열의 크기를 선언하기로 하였다.


그 후 int형 배열에 입력받은 단어를 char형으로 대문자로 변경하고 A는 int형 0번째 요소로 넣기 위해 ‘A’를 빼서 0부터 26까지 단어를 인덱스화 시켰다.

인덱스화 된 각각의 요소를 증가함으로 어떤 단어가 얼마만큼 쓰이는지 카운트 할 수 있었다.


또한 최대 사용값이 같은 알파벳이 여러개일 경우를 생각하여 답으로 나올 result 변수에 미리 여러개일 경우를 넣어두고 후에 최대가 1개이면 그 값을 다시 char형으로 변환하는 방법으로 ,최대값이 1개 초과인경우는 그대로 ‘?’를 담도록 하였다.
