Author: P_Jinsu Date: September 25, 2019  3:07:24 Category: 웹_프로그래밍

HTML문서 구조

<!DOCTYPE html>

HTML5문서를 나타냄

html문서의 내용은 <html></html>안에 작성되어야 한다.

눈에 보여질 부분은 <body></body>태그 안에 넣어주고

눈에 보이진 않으나 설정값과 관련한 부분은 <head></head>안에 넣어준다.

meta tag

해당 태그는 문서 내용이 html text 파일이고 문자 인코딩이 utf-8을 사용함을 선언한다.

meta 태그는 <head>태그 안에 들어간다.

meta 태그에서 선언한 문자 인코딩은, 소스파일 실제 인코딩과 일치하지 않으면 웹 브라우저에서 한글이 제대로 출력되지 않음.

title tag

웹 브라우저의 제목을 설정하는 태그

h1 tag

큰 제목에 해당하는 문자열 출력 태그

h1~h5까지 존제

p tag

한 단락(paragraph)에 해당하는 문자열을 p태그로 묶음

block tag / inline tag

block tag

h1 태그와 p태그 처럼 앞뒤로 줄바꿈이 저절로 일어나고 한줄을 전부 차지하는 태그를 말한다.

inline tag

한줄을 다 차지하지 않고, 앞 뒤로 줄바꿈이 저절로 일어나지 않는 태그

공백과 줄바꿈

html문서가 웹브라우저에 표시될 때, 공백 문자와 줄바꿈 문자는 무시된다.

연속되는 공백 문자는 하나로 출력하고 줄바꿈 문자도 공백 문자 하나로 출력된다.

공백 문자와 줄바꿈 태그

&nbsp;

문서에서 원하는 곳에 공백 문자 추가

<br>

문서에서 원하는 곳에 줄바꿈 문자 추가
