Author: P_Jinsu Date: September 25, 2019  4:17:36 Category: 웹_프로그래밍

연습문제

block 태그와 inline 태그의 차이는 무엇인가?

block 태그는 자동 줄바꿈이 일어나고 한 줄을 전부 차지한다.

inline 태그는 자동 줄바꿈이 일어나지 않는다.

inline 요소는 height, width가 적용되지 않음

block 요소는 vertical-align과 text-align이 적용되지 않음

block 태그에 해당하는 태그들을 나열하라.

h1, p, div, form,

inline 태그에 해당하는 태그들을 나열하라.

a, span, botton, textarea

html 문서가 웹브라우저 창에 표시될 때, 공백 문자와 줄바꿈 문자는 어떻게 표시되는가?

여러개의 공백문자는 하나의 공백문자로 줄바꿈 문자도 공백으로 표시된다.

여러 줄바꿈 문자도 공백문자로 표시된다.

HTML 문서에서 원하는 곳에서 줄바꿈이 일어나게 하려면 어떤 태그를 사용해야 하는가?

<br>태그 사용

HTML 문서에서 원하는 곳에 공백 문자를 여러 개 넣으려면 어떤 태그를 사용해야 한다.

&nbsp; 사용

nbsp = non-breaking space

html 문서에서 표를 그릴 때 사용하는 태그는 무엇인가?

<table>

표에서 한 행에 해당하는 태그는?

<tr>

표에서 한 칸에 해당하는 태그는?

<td>

html 문서의 여러 태그에 동일한 서식(style)을 일괄 적용하는 방법은 무엇인가?

<style></style>태그 사용

CSS 실렉터란 무엇인가?

<style></style>태그 안에서 서식을 지정할 태그를 지정한 것

인라인 서식 지정이란 무엇인가?

태그안에 style애트리뷰트 값으로 서식을 지정

태그의 배경색을 지정하는 서식은 무엇인가?

background-color

태그의 글자색을 지정하는 서식은 무엇인가?

color

태그의 폭을 지정하는 서식은 무엇인가?

width

태그 내부 문자열의 좌우 정렬을 지정하는 서식은 무엇인가?

text-align

태그 내부 여백을 지정하는 서식은 무엇인가?

padding

td 태그들 사이의 여백을 제거하기 위한 서식은 무엇인가?

border-collapse
