[웹프로그래밍]html태그와CSS서식정리-2

Author: P_Jinsu Date: September 25, 2019  3:37:10 Category: 웹_프로그래밍

table 태그

table 태그

기본 태그 설명

<table></table>태그 로 표를 만든다.

<tr></tr>태그 는 table row 한 행에 해당되고 한 행안에 <td></td> 태그를 사용하여 태그 안에 내용을 넣어준다.

thead, tbody 태그

표의 제목 행과 내용행을 구별하기 위해서 tr태그 위에 thead, tbody를 사용할 수 있다.

style 애트리뷰트

태그에 style 애트리뷰트를 정의하여 해당 태그의 서식을 지정할 수 있다.

글자체, 글자 크기, 글자 색, 배경 색, 여백 등을 서식으로 지정할 수 있다.

CSS 서식

Style 태그

여러 태그에 동일한 서식을 적용해야 할때, style 애트리뷰트를 사용하면 여러 태그에 반복으로 정의하기 때문에 불편, 따라서 style 태그로 일괄 적용하면 간단하다.

css 실렉터 (CSS selector)

style 태그 내용에서 서식을 적용할 태그를 지정하는 부분을 css 실렉터라고 한다.

다음과 같은 경우 td태그에 모두 서식이 적용된다.

인라인 서식

태그에 style 애트리뷰트 값으로 서식을 지정하는 것을 말한다.

인라인 서식의 우선순위

style태그의 서식보다 우선순위가 높다.

서식의 우선순위

다음과 같은경우 tr태그에 적용된 서식이 우선순위가 높아 thead아래의 tr은 tr의 내용이 적용됨

우선순위

  1. 인라인 서식

  2. css실렉터 자식태그중 가장 낮은 것부터

  3. 부모 태그 등등

:nth-child(n) 실렉터

css 실렉터 td 뒤에 nth-child를 사용하여 n번째 에 해당하는 태그를 모두 지정한다.

다음과 같을땐 2에 해당하는 곳 모두에 적용된다.

서식의 상속

모든 자식 태그에 동일한 서식은 부모 태그에 정의하면 자식태그에 상속된다.

부모 태그에 서식을 지정하고 자식태그에서 재정의(override)가 가능하다.

ex) tr태그에 정의한 내용은 td태그에도 적용된다.

:hover 실렉터

마우스 포인터 바로 아래 태그 지정

css실렉터의 공백 문자

tbody아래 모든 tr태그에 서식을 적용하는 의미

태그의 class 애트리뷰트와 css실렉터

몇몇 태그들에만 서식을 부여할때 class 애트리뷰트를 사용.

div, a {}실렉터는 div와 a모두에 적용

div a{} 실렉터는 div아레 a에 적용

div.a{} 실렉터는 div에서 a class를 사용하는 태그에 적용

다양한 CSS 서식

border

태그의 태두리를 설정하는 서식

width

태그의 가로폭

height

태그의 세로폭

text-align

left, center, right 중 하나를 줄 수 있고 Default 값은 left/ 태그 사이 내용이 서식 방향 정렬

background-color

태그의 배경색 정의

color
태그 내용에 해당하는 글자 색 정의

font-weight
글자의 굵기로 bold와 normal중 하나를 사용하고 Default값은 normal이다.

border-collapse

table 태그의 border-collapse 서식 값이 collapse이면 내부 td 태그들 사이의 여백이 제거되어 붙는다.

padding
태그의 경계선과 내부 내용 사이의 여백을 정의
