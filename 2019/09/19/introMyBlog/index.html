블로그 소개글

해당 블로그는 작성자의 문제풀이, 프로젝트, 공부 지식으로 조금씩 채워지는 공간입니다.

  • 문제풀이의 경우는 주로 JAVA로 프로그래밍 되어 예제와 풀이 방법이 올라올 예정입니다.

  • 학교에서 진행되는 튜터링과 SM사업 강의도 해당 블로그에 자료로 만들어 올립니다.

  • 프로젝트는 진행되는 사항이나 알아둬야할 지식등을 공유하기 위해 올립니다.

사용하는 프로그램 정리

목적 사용하는 프로그램
JAVA 프로그래밍 IntelliJ IDEA, Eclipse
Python 프로그래밍 Pycharm
DJango Project Pycharm, VScode
C, C++ 프로그래밍 Visual Studio2017
사진 편집 Photoshop CC 2019
동영상 편집 Premiere Pro, Vegas, 다음팟인코더
