[웹프로그래밍]URL

Author: P_Jinsu Date: September 24, 2019  1:40:51 Category: 웹_프로그래밍

URL

sw.skhu.ac.kr 과 skhu.ac.kr/sw의 차이

sw.skhu.ac.kr

서버 주소 자체이고 웹 서버 컴퓨터 서버 주소이다.

장점 : 주소가 간결함

단점 : 다수의 경우에는 주소가 많아진다.

skhu.ac.kr/sw

sw가 주소에 포함된 어플리케이션

장점 : 관리가 간단하다.

단점 : 주소가 복잡해진다.

URL의 예

https://comic.naver.com:8080/webtoon/detail.nhn?titleId=20853&no=1193

http

통신 프로토콜

comic.naver.com

웹서버 컴퓨터의 주소

8080

웹서버가 인터넷 TCP/IP통신에 사용할 포트(port) 번호.

디폴트 번호는 80이고 디폴트의 경우 생략한다.

webtoon

context 이름, context path

context 이름 : 웹서버에서 실행되는 웹 어플리케이션의 이름

eclipse에서 개발시 project이름이 톰캣 웹서버에서 실행될때 context 이름이 된다.

detail.nhn

문서의 이름

?titleId=20853&no=1193

query String, 문서의 경로명 뒤에 추가 파라미터 데이터.

웹서버에서 동적 문서가 생성되고 수정 될 때 사용.
