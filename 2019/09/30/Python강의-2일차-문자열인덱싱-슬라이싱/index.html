추가개념

Docstring

클래스, 함수, 모듈에서 첫 번째 표현식으로 나타나는 문자열 리터럴(읽기 전용 메모리에 할당한 문자열). 스위트(들여쓰기 후 있는 코드 행 하나)가 실행 될때는 무시되지만, 컴파일러에 의해 인지되어 둘러싼 클래스,함수, 모듈의 __doc__ 어트리뷰트(attribute,속성)로 삽입된다. 인트로스팩션([introspection, 객체정보)을 통해 사용 할 수 있으므로, 객체 설명서를 위한 규범적인 장소.

간단하게 말해서 코드의 문서화에 도움이 되는 문자열, Python에서 함수 몸체 시작 부분에 문자열을 포함 시켜서 함수 정의에 문서를 붙일 수 있다.


Python에서 Docstring은 “”” “””으로 사용하고 이것으로 여러줄의 주석으로 사용도 가능하며

모듈 파일 처음이나 함수, 클래스 선언후 다음라인에 작성하며, 중요 내용을 docstring 첫라인에 적고 한깐 띄고 자세한 내용을 적는 것이 표준이다.

문자열끼리 연산

Python에서는 문자열을 숫자처럼 더하고 곱할 수 있다.

  • +연산은 두 문자열을 서로 연결시키는 역할을 한다.
  • *연산은 해당 문자열을 정수배 반복하여 출력해준다.

문자열 인덱싱과 문자열 슬라이싱

문자열 인덱싱

문자열을 배열의 인덱스로 다룰 수 있게 해준다.

문자열은 문자들의 집합(배열)이라 볼 수 있어, Python에서는 문자열 내에서 특정 위치의 문자를 쉽게 선택할 수 있도록 문자열을 문자들의 배열로 다룰 수 있다.

다음과 같이 문자열의 첫 문자는 index 0부터 시작해서 1씩 증가된다.

또한 공백(white space)도 하나의 문자로 취급된다.

파이썬 인덱싱에서 추가 개념

Python은 음수 인덱스도 지원한다.

음수 인덱스의 경우 마지막 문자를 -1로 생각하고 역순으로 증가한다.

index 0 1 2 3 4 5 6 7 8
value 파 이 썬 문 자 열 ㅎ ㅎ

문자열 슬라이싱

문자열을 다루다보면 문자열의 일부분만 사용하거나 변경할 경우가 생기는데, 이러한 경우도 Python이 쉽게 처리할 수 있도록 문자열 일부를 잘라서 반환하는 slicing을 지원한다.

:로 문자열 슬라이싱을 이용할 수 있고 [시작 번호:끝 번호]로 슬라이싱이 가능하다.

다음과 같은 방법을 수식으로 표현하면
$$
start <= st < end
$$
이므로 start = 1이고 end=4이면 해당 문자는 [1],[2],[3]이다.

사용하거나 외울때 [어느 인덱스부터:어느 인덱스 전까지]로 외우는 것이 생각하기 편하다.

또한 앞번호를 생략하면

처음부터 end index전까지 반환되고

뒷번호를 생략하면

start index부터 마지막까지 반환
