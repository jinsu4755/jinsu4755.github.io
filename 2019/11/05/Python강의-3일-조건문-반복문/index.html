조건문

조건문이란?

주어진 조건 식의 결과에 따라 다른 명령을 수행하도록 프로그램의 흐름을 제어하는 명령문

###

if 문

if문 단독

if문은 위와 같이 단독으로 사용이 가능하다.

조건식의 결과가 True 이면 if문 다음에 명령문들이 실행된다.

만약, 조건식이 False일때 작동하는 명령문을 사용하고 싶다면 아래와 같이 사용할 수 있다.

if-else문

단, if문은 단독으로 사용이 가능하지만 else문은 단독으로 사용할 수 없으며 반드시 if문과 함께 사용해야한다.

반복문

반복문?

반복적인 작업을 특정 조건을 만족시킬 때까지(다른말로 해당 조건이 만족하는 동안) 행동을 지속적으로 반복해서 수행하는 문법

while문

while문은 조건식을 주어주고 조건이 만족하면 반복하는 반복 명령문이다.

해당 반복문 안에 a의 크기를 증가시키는 명령문이 있으면 a가 5일때 반복문을 탈출하여 더이상 while문에 진입하지 않는다.

while 문에서 가장 중요한 것은 해당 while문을 탈출시키는 탈출 조건이다.

만약, 탈출조건이 작성되지 않거나 잘못 쓰인경우 해당 반복문은 무한 루프하게 된다.

따라서, while 반복문은 특별히 의도하여 무한루프를 시키는 것이 아니라면 탈출 조건을 반드시 사용해야한다.
