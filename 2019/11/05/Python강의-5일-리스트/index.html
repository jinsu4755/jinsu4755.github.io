C언어에서의 배열과 유사하다.

python에서 리스트는 아래와 같은 특징을 가진다.

  1. 리스트에 저장요소가 모두 같은 타입일 필요가 없다.
  2. 요소들은 순서대로 저장하며, 0부터 시작하는 인덱스로 접근이 가능하다.
  3. 리스트는 값 변경이 가능하다
