Author: P_Jinsu Date: November 5, 2019  21:46:44 Category: Python강의/SM사업

4일차

함수

함수(function)

프로그래밍에서 하나의 특정 작업을 수행하기 위해 독립적으로 설계된 프로그래밍 코드의 집합을 말한다.

주로 특정 arguments 인자들을 받아서 return으로 연산 결과를 반환하거나 작업을 수행한다.

인자(arguments)는 함수 호출시 내부에서 사용할 수 있도록 데이터를 전달하는 변수를 말한다.(=매개변수)

함수를 사용하는 이유?

지금까지 print(), range(), input() 모두 이미 python에서 제공하는 함수이다.

프로그램 내에서 중복적인 코드의 작성을 최소화 하고 코드의 재사용성을 높이기 위해서 사용한다.

함수를 사용하지 않으면 print, range전부 다 다시 정의해서 사용해야한다.

return

위 예제 처럼 함수에서 사용한 명령문의 결과를 return 문을 사용하여 반환이 가능하다.
