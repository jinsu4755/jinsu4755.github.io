[웹프로그래밍]뉴스페이지만들며개념보충연습문제

Author: P_Jinsu Date: October 2, 2019  2:07:24 Category: 웹_프로그래밍

웹페이지에 사진을 표시하기 위한 태그는 무엇인가?

img태그

1.5 배 줄간격을 지정하는 방법은?

line-height 서식

태그들 사이의 여백을 지정하는 서식은 무엇인가?

margin

네이버 웹페이지처럼 웹페이지의 내용이 웹브라우저 창 가운데 표시되게 하려면?

margin값으로 left와 right 모두 auto값을 지정한다

어떤 태그를 특정 x, y 좌표에 표시되게 하려면?

left와 top을 사용한다.

어떤 태그에 표시될 폰트를 지정하는 방법은?

font 서식 사용

float: right; 서식을 적용 받은 태그는 어떻게 그려지는가?

다른 태그들과 같은 줄에 표시되지 않고 왼쪽으로 치우쳐 따로 그려진다.

position: absolute; 서식을 적용 받은 태그는 어떻게 그려지는가?

다른 태그들 위에 떠있는 형식으로 그려진다.

아래 서식은 어떠 태그에 적용되는가?

tbody tr:nth-child(2) { color: red; }

tbody 아래 tr태그중 2번째에 적용

div.menu span { color: read; }

div에서 class애트리뷰트로 menu를 가진 태그 아래 span에서 적용

tr.odd td:nth-child(1) { color: red; }

tr에서 class애트리뷰트로 odd를 가진 td태그 1번째에 적용
