[웹프로그래밍]뉴스 페이지 만들며 개념보충

Author: P_Jinsu Date: October 2, 2019  1:25:54 Category: 웹_프로그래밍

태그와 서식 보충하기

line-height 서식

줄 간격 지정

line-height: 180% 이면 줄간격 180%

img 태그

<img src="/html1/images/html5.png">

현재 html 문서에 이미지 파일 내용을 표시할 때 사용된다.

img태그에 src애트리뷰트는 이미지 파일의 URL을 나타낸다.

웹브라우저는 즉시 src 애트리뷰트 URL을 웹서버에 요청한다.

src = “/html1/images/html5.png”

기존 URL주소에서 서버 주소 부분이 생략됨, 일부가 생략된경우 현재 html문서에 URL에서 해당부분을 그대로 사용한다.

현재 html 문서의 URL이 http://localhost:8080/html1/03/news03.html 이므로

이미지 파일의 URL은 http://localhost:8080/html1/images/html5.png가 된다.

웹브라우저는 <img>태그를 그리기 위해서 이미지 URL을 웹서버에 요청하고

웹서버는 요청 받은 이미지 파일을 웹브라우저에 전송한다.

웹 서버는 전송 받은 이미지 파일을 img 태그 위치에 그려넣는다.

float 서식

해당 서식을 부여받은 태그는 다른 태그들과 같은 줄에 표시되지 않고 left, right등 속성값 방향으로 치우쳐 따로 그려진다.
