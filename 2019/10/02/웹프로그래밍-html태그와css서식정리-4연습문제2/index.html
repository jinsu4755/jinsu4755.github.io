[웹프로그래밍]html태그와css서식정리-4연습문제2

Author: P_Jinsu Date: October 2, 2019  1:17:59 Category: 웹_프로그래밍

태그의 폭을 지정하는 서식은 무엇인가?

width 서식

태그의 높이를 지정하는 서식은 무엇인가?

height 서식

태그 내부 문자열의 수평 방향 정렬을 지정하는 서식은 무엇인가?

text-align

태그 내부 문자열의 수직 방향 정렬을 지정하는 서식은 무엇인가?

vertical-align

다음 서식은 각각 어떤 태그들에 적용되는가?

p span { color: red; }

p태그 아래 span태그에 적용된다.

p, span { color: red; }

p태그와 span태그 모두 적용된다.

p.s { color: red; }

p태그에서 class 애트리뷰트에 s속성을 준 태그에서 적용.
