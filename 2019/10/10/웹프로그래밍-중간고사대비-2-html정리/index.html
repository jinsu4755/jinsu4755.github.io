[웹프로그래밍][중간고사대비]2.html정리

Author: P_Jinsu Date: October 10, 2019  20:29:41 Category: 웹_프로그래밍

html tag 정리

html tags

html 문서의 구조

<!DOCTYPE html>

html 5 문서를 나타낸다.

html 문서 내용은 <html></html>안에 작성되어야하고

눈에 보여질 부분은 <body></body>에 작성된다.

눈에 보이지 않지만 설정 값과 같은 것들은 <head></head>내부에 선언한다.

<meta>

meta tag는 해당 태그의 문서 내용과 문자인코딩을 선언할 때 사용한다.

다음과 같이 사용한다.

<title>

title tag는 주로 해당 문서가 표시될 웹 브라우저의 제목을 성정하는 태그이다.

다음과 같은 경우 웹페이지 제목이 hello가 된다.

<h1>, <h2>, <h3> 등 h태그

큰 제목에 해당하는 문자열 출력 태그

h1~h5까지 존재한다.

<p></p>

p tag는 한 단락(paragraph)에 해당하는 문자열을 p태그로 묶어서 화면에 출력한다.

<a>

a tag는 링크를 걸어줄때 사용한다. 링크 내용은 href=”주소”로 연결 할 수 있다.

<table></table>

table태그는 html문서에 표 (table)을 그릴때 사용한다.

<tr></tr>

tr태그는 table row의 약자로 한 행에 해당되고 행안에 td태그를 통해 데이터를 넣는다.

<td></td>

td 태그는 표안에 내용을 채워주는 역할을 한다.

<thead></thead>

표의 제목을 나타낼때 사용되는 태그

<tbody></tbody>

표의 내용을 나타낼때 사용되는 태그

두 태그는 표에서 제목부분과 몸체 부분을 나누어 표현할때 사용된다.

<style></style>

style 태그는 여러 태그에 동일한 css서식을 적용해야 할 때 사용된다.

<img>

img 태그는 html 문서에 이미지를 표시할때 사용한다.

src =”이미지 주소” 로 기존 URL주소에서 서버 주소 부분이 생략된 해당 부분을 그대로 사용한다.

<div></div>

div 태그는 division의 약자로 웹사이트의 전체적인 레이아웃을 만들때 사용한다.

Block tag and Inline tag

block tag

h1, p, div, form 처럼 앞 뒤로 줄바꿈이 저절로 일어나고 한 줄을 전부 차지하는 태그를 말한다.

inline tag

span, button, a, textarea 처럼 한 줄을 모두 차지하지 않고, 앞 뒤로 줄바꿈이 일어나지 않는 태그들을 말한다.

html 문서에서 공백과 줄바꿈

&nbsp;

문서에 원하는 곳에서 공백 문자를 추가해준다.

<br>

문서에서 원하는 곳에 줄바꿈 문자를 추가한다.

자주 쓰이는 html 특수문자.

html 특수 문자문자&nbsp;공백&gt;>&lt;<&amp;&

html 특수 문자 문자
&nbsp; 공백
&gt; >
&lt; <
